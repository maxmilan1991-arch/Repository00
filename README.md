# Repository00

This repository hosts **immobiliare-export**, a Python CLI tool that
scrapes real-estate listings from immobiliare.it into a local SQLite
archive and exports them to a styled multi-sheet `.xlsx` report.

The implementation lives under [`immobiliare_export/`](./immobiliare_export/).
See its [`README.md`](./immobiliare_export/README.md) for usage,
configuration, and architecture notes.
